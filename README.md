## Install

you must use [elm-github-install](https://github.com/gdotdesign/elm-github-install) and include in your elm-package.json as follows:

```
"dependencies": {
  "kingsleyh/elm-neo": "1.0.5 <= v < 2.0.0"
}
```

Then run elm-install
